# Data Structure

## BIT / Segment Tree manipulation supports

| class                  | 設值範圍 | 加值範圍 | 查詢區間和 | 查詢區間極值 | 查詢自訂區間 <br/> (lcm, gcd, etc) |
| ---------------------- | :------: | :------: | :--------: | :----------: | :--------------------------------: |
| BIT                    |   單點   |   單點   |     可     |     不可     |                不可                |
| RangeUpdateBIT         |   單點   |   區間   |     可     |     不可     |                不可                |
| BIT2D                  |   單點   |   單點   |     可     |     不可     |                不可                |
| SegmentTree            |   單點   |   單點   |     可     |      可      |                 可                 |
| RangeUpdateSegmentTree |   區間   |   區間   |     可     |      可      |                不可                |
